# Lahore Route Navigator

A graph-based route navigation system for Lahore that visualizes multiple route options and dynamically highlights the shortest path.

## Features

- Select source and destination from dropdown menu
- Visual graph-based node representation
- Three route options displayed (Orange, Blue, Purple)
- Purple route represents the shortest path
- Direction cards showing route details
- Clean black-themed UI

## Technical Stack

- HTML
- CSS
- JavaScript (UI & interaction)
- C++ (route data building and graph logic)

## How It Works

Locations are modeled as graph nodes and routes as weighted edges.
The system computes possible paths and identifies the optimal shortest route,
which is visually highlighted in the interface.
